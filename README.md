# T's — AI Email Assistant

A modern AI-powered email assistant built with **HTML5**, **CSS3**, **Bootstrap 5**, and **Vanilla JavaScript (ES6)**. Generate, reply to, improve, rewrite, and summarize emails through a clean chat-style interface using any OpenAI-compatible AI API.

---

## Features

- Generate, reply, rewrite, improve, and summarize emails
- Chat-style AI conversation
- Chat history with local storage
- Copy and regenerate responses
- Export conversations as TXT
- Dark & Light mode
- Fully responsive design
- Supports Groq, OpenRouter, OpenAI, and local AI models
- Browser-based settings and history

---

## Tech Stack

- HTML5

- Bootstrap 5
- Vanilla JavaScript (ES6)
- Bootstrap Icons

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Tayyaba-Ajmal/ai-email-assistant.git
```

Open **index.html** in your browser or run a local server:

```bash
python3 -m http.server
```

Then open:

```
http://localhost:8000
```

Enter your API key Get it from any openAI resources and choose **Settings**, choose a provider, and start generating emails.
